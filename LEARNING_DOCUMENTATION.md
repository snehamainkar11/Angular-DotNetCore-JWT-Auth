# Learning Documentation

## Learning Objectives
- Understand the fundamentals of Angular and .NET Core.
- Implement JWT Authentication in web applications.
- Develop full-stack projects using Angular and .NET Core.

## Concepts Covered
- Angular Components, Services, and Modules.
- .NET Core MVC Architecture.
- RESTful APIs and Client-Server Communication.
- User Authentication and Authorization with JWT.

## Skills Gained
- Proficiency in building scalable web applications.
- Experience in using TypeScript and C#.
- Knowledge of integrating front-end and back-end technologies.

## Tech Stack Exploration
- **Front-end:** Angular, TypeScript, HTML, CSS
- **Back-end:** .NET Core, C#
- **Database:** SQL Server, Entity Framework

## Learning Path
1. Basics of HTML, CSS, JavaScript
2. Introduction to TypeScript
3. Learning Angular fundamentals
4. Understanding ASP.NET Core basics
5. Implementing Authentication and Authorization
6. Building a full-stack application

## Key Learnings
- The importance of understanding both front-end and back-end development.
- How JWT helps in secure communication between the client and server.
- The value of continuous learning and practical application of skills acquired in real-world projects.
